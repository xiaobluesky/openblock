### 2. Obsidian (~15k ⭐)

**项目地址:** https://obsidian.md (闭源商业产品)  
**GitHub:** 官方插件仓库 ~15k

**蹭流量策略:**

#### A. 差异化定位
```
Obsidian 优势:
✅ 本地 Markdown 文件
✅ 丰富的插件生态
✅ 双向链接

OpenBlock 优势:
✅ 块编辑器 (Obsidian 是纯 Markdown)
✅ 更直观的拖拽体验
✅ 完全免费 (Obsidian 同步付费)
✅ 中文原生优化
```

#### B. 内容营销
```
文章标题:
1. "Obsidian 用户为什么需要块编辑器？"
2. "从 Obsidian 迁移到 OpenBlock 的 5 个理由"
3. "Obsidian 同步要钱？这个开源方案免费"

发布平台:
- Obsidian 官方论坛 (谨慎，避免被删)
- Reddit r/ObsidianMD (谨慎)
- 知乎"Obsidian 好用吗"问题下回答
```

#### C. 技术对比
```
对比维度:
- 编辑器体验: 块编辑 vs Markdown
- 学习曲线: OpenBlock 更陡峭 (但更强大)
- 数据格式: OpenBlock (JSON+MD) vs Obsidian (纯 MD)
- 同步成本: OpenBlock (免费 WebDAV) vs Obsidian ($8/月)
```

**定位:** Notion 的开源替代品 (Rust + Flutter)

**蹭流量策略:**

```

❌ 较重的技术栈
❌ 中文支持一般
❌ 社区活跃度下降

✅ 中文原生优化
```


#### C. 内容对比
