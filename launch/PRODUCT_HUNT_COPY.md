### Description (详细描述)
```
🏠 Local-First Knowledge Workspace

OpenBlock gives you the best of Notion (block editor, rich formatting, page trees) with the privacy of Obsidian (local storage, open formats, self-hosted sync).

✨ Why OpenBlock?

• 🏠 Local-first: Works offline, your data never leaves your device unless you want it to
• 🔓 Open formats: Markdown + JSON, export anytime, no vendor lock-in
• 🔄 Optional sync: Self-host with WebDAV/S3/WebRTC, full control
• 🇨🇳 Chinese optimized: Native support, no latency for Chinese users
• 🛡️ Privacy first: No data collection, optional end-to-end encryption
• 🧩 Plugin system: Open plugin architecture, extend freely
• 💰 100% Free: MIT licensed, no hidden costs, no premium tiers

🆚 How does it compare?

| Feature | OpenBlock | Notion | Obsidian |
|---------|-----------|--------|----------|
| Block Editor | ✅ | ✅ | ❌ (Markdown) |
| Offline First | ✅ | ❌ | ✅ |
| Open Data Format | ✅ | ❌ | ✅ |
| Self-Hosted Sync | ✅ | ❌ | ⚠️ (Paid) |
| Chinese Optimized | ✅ | ⚠️ | ⚠️ |
| Free | ✅ | ❌ | ✅ |
| Open Source | ✅ | ❌ | ❌ |

🚀 Quick Start

```bash
# From source
git clone https://github.com/昊尘 002/openblock.git
cd openblock
npm install
npm run dev

# Docker (coming soon)
docker run -d -p 3000:3000 -v ./data:/app/data openblock/openblock:latest
```
